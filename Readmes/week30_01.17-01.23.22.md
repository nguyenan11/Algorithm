## [Leetcode #55 - Jump Game](https://leetcode.com/problems/jump-game/)

#### Level: Medium ðŸ“˜

> Greedy approach / backward
```python
def canJump(self, nums: List[int]) -> bool:
  target = len(nums) - 1 # last index in the array
  for i in range(len(nums) - 1, -1 , -1):
    if i + nums[i] >= target:
      target = i
  return target == 0
```

### O(n) time | O(1) space
