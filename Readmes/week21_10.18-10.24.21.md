## [Leetcode #412. Fizz Buzz](https://leetcode.com/problems/fizz-buzz/)

#### Level: Easy ðŸ“— (very)

```python
def fizzBuzz(self, n):
  """
  :type n: int
  :rtype: List[str]
  """
  result = []
  for i in range(1, n + 1):
    curr_word = ""
    if i % 3 == 0:
      curr_word += "Fizz"
    if i % 5 == 0:
      curr_word += "Buzz"
    if not curr_word:
      curr_word = str(i)
    result.append(curr_word)
  return result
```

###  O(n) time | O(1) space
