## [Leetcode #412. Fizz Buzz](https://leetcode.com/problems/fizz-buzz/)

#### Level: Easy ðŸ“— (very)

```python
def fizzBuzz(self, n):
  """
  :type n: int
  :rtype: List[str]
  """
  result = []
  for i in range(1, n + 1):
    if i % 15 == 0:
      result.append("FizzBuzz")
    elif i % 3 == 0:
      result.append("Fizz")
    elif i % 5 == 0:
      result.append("Buzz")
    else:
      result.append(str(i))
  return result
```

###  O(n) time | O(1) space
